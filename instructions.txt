üöÄ How to Run the Project

Follow the steps below to set up and run the Student Management System locally:

1. Install Required Software
   Make sure the following are installed on your system:
   - Node.js
   - npm (comes bundled with Node.js)
   - MongoDB

2. Ensure You Are Connected to the Internet
   Some dependencies may be fetched online during installation.

3. Navigate to the Source Code Directory
   Open your preferred terminal and run:
   cd Group 2 - Student Management System/Source Code/student-mgmt-system/

4. Install Dependencies
   Use npm to install all required node modules:
   npm install

5. Start the Development Server
   Once installation is complete, run the project with:
   npm run dev

6. Access the Application
   Open any web browser and go to:
   http://localhost:3002/

   The server will be running on port 3002.
